## 📦 Package Manager: pnpm

**IMPORTANTE:** Este proyecto utiliza **pnpm** como gestor de paquetes oficial. Debes usar pnpm para instalar dependencias.

```bash
# ✅ CORRECTO - Usar pnpm
pnpm install
pnpm start
pnpm build

# ❌ INCORRECTO - No usar npm o yarn
npm install       ❌
yarn install      ❌
```

Para más información sobre pnpm, lee [PNPM-GUIDE.md](./PNPM-GUIDE.md).
